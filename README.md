**Doodly**

A lightweight drawing app that runs entirely in your browser.

**Features**

- brush
- eraser
- circle shape
- rectangle shape
- line shape
- paint color picker
- tool size controls

- layer area select
- copy selection
- cut selection
- paste selection

- background color picker
- export area select
- export padding
- export include background
- download export

- infinite canvas with pan & zoom
- multiple layers with image import support
- reorder layers
- change layer visibility
- convert layer to image and vice versa
- delete layer
